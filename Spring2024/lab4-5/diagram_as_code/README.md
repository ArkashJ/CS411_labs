# Run this code:

## Setup

```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt

```

## Install Graphviz to render the diagram

```bash
brew install graphviz
```

## Run

```bash
python3 diagram_code.py
```
